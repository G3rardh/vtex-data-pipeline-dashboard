# Tabla Sucursales

## Para qué sirve

Tabla maestra que contiene las sucursales oficiales para validar warehouses y configuraciones.

## De dónde salió la data

Exportación del sistema DAD.

## Qué validé

* sucursales duplicadas
* estados activos/inactivos
* cruce con warehouses VTEX

## Dónde se cargó

Base de datos PostgreSQL.

## Para qué se usará

Validaciones y dashboards logísticos.
